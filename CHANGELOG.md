## [1.1.0] - 2026-02-14

### Added
- Multiple TV profiles
- Profile editor with add/edit/delete controls
- Power status indicator with dynamic button coloring
- App launcher (load + start)
- Input switching (load + switch)
- GitHub footer link

### Changed
- Sidebar UI layout improvements
- Icon-only buttons for profile management
- Navigation button reordering
- Channel and volume button layout adjustments
- Apps and Inputs controls moved next to select fields

### Fixed
- Bravia API "Illegal JSON" compatibility issues
- Improved JSON-RPC version handling
- IRCC code auto-detection with fallback
